# Sistema de Login (Base)

Este projeto é uma base simples para um sistema de autenticação de usuários em um site.
Ele contém a estrutura inicial de uma tela de login e a lógica básica de validação.

⚠️ Este projeto ainda está em desenvolvimento.

## Funcionalidades

- Tela de login
- Campo de usuário
- Campo de senha
- Validação básica
- Estrutura pronta para integração com banco de dados

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
